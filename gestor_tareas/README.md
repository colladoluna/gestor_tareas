# Gestor de Tareas

Aplicación web para gestionar tareas hecha con Python y Flask.
La base de datos es SQLite y se crea sola al ejecutar el programa.

## Tecnologías usadas

- Python
- Flask
- SQLite
- HTML y CSS

## Cómo ejecutarlo

Instalar Flask:
pip install flask

Ejecutar la aplicación:
python app.py

Abrir en el navegador:
http://localhost:5000

## Funcionalidades

- Añadir tareas
- Editar tareas
- Eliminar tareas
- Marcar como completada
- Filtrar por estado y prioridad
